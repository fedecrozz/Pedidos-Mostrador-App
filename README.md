# Pedidos Mostrador App

Aplicación SaaS Web para mostrar el estado de los pedidos en restaurantes.

## Estructura
- Frontend: React
- Backend: Node.js + Express
- Base de datos: SQLite

## Funcionalidades principales
- Panel principal con todos los pedidos y su estado
- Botón para cambiar estado de pedido
- Botón para eliminar pedido

## Instalación y ejecución

### Backend
1. Instalar dependencias:
   ```bash
   cd backend
   npm install
   ```
2. Iniciar servidor:
   ```bash
   npm start
   ```
