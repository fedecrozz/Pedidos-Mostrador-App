# Ícono de la aplicación

Este directorio contiene los íconos para la aplicación.

Para crear un ejecutable con ícono personalizado:

1. Coloca tu ícono en formato `.ico` (para Windows) como `icon.ico`
2. Opcionalmente, añade `icon.png` (512x512px) para otros formatos

Si no tienes un ícono personalizado, puedes crear uno en:
- https://www.icoconverter.com/
- https://favicon.io/

El ícono debe ser de 256x256 píxeles o mayor para mejores resultados.
